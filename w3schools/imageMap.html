Mapas con imágenes

Descripción

La etiqueta <map></map> permite definir un mapa de enlaces dentro de una imagen. Esto significa que la imagen puede ser cliqueada en distintas zonas y dirigir a distintas páginas (o a ninguna, si se cliquea en una zona sin asignación). Para definir cada zona que llevará a un enlace diferente, se utiliza la etiqueta <area>. La etiqueta map tiene apertura y cierre, mientras que area es etiqueta simple.

Para crear un mapa dentro de una imagen primero debe agregarse la imagen con la etiqueta img, como lo hemos hecho hasta ahora, con los atibutos src y alt, pero adicionalmente se debe agregar el atributo usemap="#nombreMapa" agregando el nombre de nuestro mapa antecedido por un síbolo #.

Posteriormente, dentro de las dos etiquetas map se deben agregar las áreas que queremos definir con la etiqueta <area>. La etiqueta área debe llevar los siguientes tributos:

  • shape="forma" Define la forma que debe llevar el área en el mapa. Las formas posibles son "rect" para rectángulo, "circle" para una circunferencia, "poly" para un polígono y "default" para el resto de la imagen.
  • cords="coordenadas" Define las coordenadas para el área a generar. Estas coordenadas dependen de la forma del área creada y siempre se expresan en píxeles. Para las formas rectangulares, las coordenadas son las coordenadas cartesianas x y y de dos esquinas opuestas (4 números en total) separados por comas. Para las formas circulares, se utilizan las coordenadas cartesianas del centro del circulo y su radio. Por último, para las formas poligonales se utilizan las coordenadas de cada vértice del polígono.
  • alt="alterno" Define el texto alterno a mostrarse en caso de que no esté disponible el área.
  • src="ruta" Define el enlace al que debe de ser dirigido el usuario en cada zona.

Ejemplo

Ejemplo de un mapa dentro de una imagen donde existen enlaces en cada objeto de la imagen, los cuales dirigen cada uno a un artículo distinto de Wikipedia:

<html>

<head>
<title></title>
<meta charset="UTF-8">
</head>

<body>
<img src="../images/map.jpg" alt="Escritorio de trabajo" usemap="#mapa_escritorio">
<map name="mapa_escritorio">
<area shape="rect" coords="34,44,270,350" alt="Laptop" href="https://es.wikipedia.org/wiki/Computadora_portátil">
<area shape="rect" coords="290,172,333,250" alt="Teléfono Móvil" href="https://es.wikipedia.org/wiki/Teléfono_móvil">
<area shape="circle" coords="337,300,44" alt="Café" href="https://es.wikipedia.org/wiki/Café">
</map>
</body>

</html>
