Border en CSS

Descripción

Dentro del estilo que podemos darle a un boque HTML se encuentra border, el cual, como su nombre lo indica, nos permite darle estilo al borde del bloque. Podemos modificar la forma en que se muestra, el color y el grosor. Es posible especificar lo 4 bordes por separado con la conjunción de border y left, top, bottom o right.

Ejemplo

Ejemplo de border completo:

<html>
<head>
<title> </title>
<meta charset="UTF-8">
</head>
<body>
<div style="border:2px solid blue">Este bloque tiene borde azul de 2 píxeles</div>
<div style="border:5px solid green">Este bloque tiene borde azul de 2 píxeles</div>
</body>
</html>

Vista previa:

Este bloque tiene borde azul de 2 píxeles
Este bloque tiene borde azul de 2 píxeles

Ejemplo de bordes por separado:

<html>
<head>
<title> </title>
<meta charset="UTF-8">
</head>
<body>
<div style="border-left:2px solid blue">Este bloque tiene borde izquierdo azul de 2 píxeles</div>
<div style="border-right:5px solid green">Este bloque tiene borde derecho azul de 5 píxeles</div>
</body>
</html>
