Padding

Descripción

La propiedad padding es una herramienta en CSS que sirve para especificar el espacio entre el border (borde del elemento) y su contenido, ya sea a la derecha, izquierda, arriba o abajo. En otras palabras, es el espacio inmediato interior a border. Al igual que border, puede especificarse por separado en cada uno de los cuatro lados del bloque.

Ejemplo

Ejemplo de un párrafo con 30px de padding y border de 2px, sólido y de color azul:

<html>
<head>
<title> </title>
<meta charset="UTF-8">
<style>
p
{
border: 2px solid blue;
padding: 30px;
}
</style>
</head>
<body>
<p>
Hola mundo
</p>
</body>
</html>

Ejemplo de un párrafo con 15px de padding y border de 2px, sólido y de color azul:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
p
{
border: 2px solid blue;
padding: 15px;
}
</style>
</head>
<body>
<p>
Hola mundo
</p>
</body>
</html>

Hola mundo

Puede notarse el rectángulo que encierra al primer párrafo (su border) es más grande que el que encierra al segundo. Esto se debe a que ambos tienen el mismo tamaño de fuente, pero enseguida al contenido, el primer párrafo debe mantener 30px de distancia para comenzar pintar el borde, mientras que el segundo sólo tiene que mantener 15px de distancia; esta distancia es el padding. Luego, inmediatamente despúes del padding, observamos el borde que en ambos es del mismo color y grosor.
