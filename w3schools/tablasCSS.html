Edición de tablas con CSS

Propiedad border en tablas

Descripción

La propiedad border puede ser utilizada en tablas y celdas. De esta forma, la línea que rodea cada celda o la de la tabla entera puede ser cambiada de color o de grueso.

Ejemplo

Ejemplo de tabla con borde de 3 pixeles de grueso y color naranja:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
table, th, td {
border: 3px solid orange;
}
</style>
</head>
<body>

<table>
<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
<tr>
<td>John</td>
<td>Doe</td>
<td>80</td>
</tr>
</table>

</body>
</html>

Bordes de celdas unidos

Descripción

La propiedad border-collapse: collapse; ayuda a que los bordes de las celdas no luzcan separados y en su lugar aparezcan completamente unidos.

Ejemplo

Ejemplo donde se toma la misma tabla del ejemplo anterior y se agregan los bordes colapsados:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
table, th, td {
border: 3px solid orange;
border-collapse: collapse;
}
</style>
</head>
<body>

<table>
<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>

<td>94</td>
</tr>
<tr>
<td>John</td>
<td>Doe</td>

</tr>

</body>
</html>

Estilo completo a tablas

Descripción

En general, las tablas pueden hacer uso de las propiedades que utilizan el resto de elementos html. Se pueden agregar, por ejemplo, paddings, fuentes, colores de fondo, alineación de texto, entre varias propiedades más.

Ejemplo

Ejemplo donde se toma la misma tabla del ejemplo anterior y se agregan los bordes colapsados:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
