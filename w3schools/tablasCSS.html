Edición de tablas con CSS

Propiedad border en tablas

Descripción

La propiedad border puede ser utilizada en tablas y celdas. De esta forma, la línea que rodea cada celda o la de la tabla entera puede ser cambiada de color o de grueso.

Ejemplo

Ejemplo de tabla con borde de 3 pixeles de grueso y color naranja:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
table, th, td {
border: 3px solid orange;
}
</style>
</head>
<body>

<table>
<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
<tr>
<td>John</td>
<td>Doe</td>
<td>80</td>
</tr>
</table>

</body>
</html>

Bordes de celdas unidos

Descripción

La propiedad border-collapse: collapse; ayuda a que los bordes de las celdas no luzcan separados y en su lugar aparezcan completamente unidos.

Ejemplo

Ejemplo donde se toma la misma tabla del ejemplo anterior y se agregan los bordes colapsados:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
table, th, td {
border: 3px solid orange;
border-collapse: collapse;
}
</style>
</head>
<body>

<table>

Estilo completo a tablas

Descripción

En general, las tablas pueden hacer uso de las propiedades que utilizan el resto de elementos html. Se pueden agregar, por ejemplo, paddings, fuentes, colores de fondo, alineación de texto, entre varias propiedades más. Además, pueden crearse id y clases para usarlas en más de una tabla.

Ejemplo

Ejemplo donde se da un estilo completo a una tabla; se da color a los bordes exteriores, a los bordes entre filas y al fondo de la cabecera, se le da distinto padding a las cabeceras y a los datos, la primera columna está en negritas y la última está alineada a la derecha, las filas pares son de un tono más claro que las impares. Todo el estilo está creado en una clase para ser usado sólo en algunas tablas.

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>

.tabla3, .tabla3 th, .tabla3 td{
font-family: sans-serif;
border-collapse: collapse;
}
.tabla3{
border: 1px solid rgb(52, 116, 235);
}
.tabla3 tr:nth-child(odd){
background-color: rgb(185, 207, 248);
}
.tabla3 tr:nth-child(even){
background-color: rgb(232, 239, 253);
}
.tabla3 th{
color: white;
background-color: rgb(52, 116, 235);
padding: 0.3em 2em;
border: 0px;
}
.tabla3 td{
padding: 0.1em 0.5em;
border-bottom: 1px solid rgb(52, 116, 235);
border-top: 1px solid rgb(52, 116, 235);
}
.tabla3 td:nth-child(1){
font-weight: bold;
}
.tabla3 td:nth-child(3){
text-align: right;
}

</style>
</head>
<body>

<table class="tabla3">
<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
<tr>
<td>John</td>
<td>Doe</td>
<td>80</td>
</tr>
</table>

</body>
</html>
