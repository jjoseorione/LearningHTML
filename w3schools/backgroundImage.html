Imágen de fondo con CSS

Descripción

En cualquier elemento de HTML es posible definir un color de fondo; de la misma manera, también es posible definir una magen de fondo. Para ello, se debe de usar la propuedad CSS background-image: url(ruta_imagen). Esta propiedad se puede usar en cualquier elemento HTML, aunque si se quiere usar como imagen de fondo a la página, debe usarse dentro del elemento body.

Si se utiliza sólo el elemento mencionado, la imagen abarcará todo su tamaño original y, en caso de hacer falta espacio, se repetirá hasta abarcar todo el elemento. Para evitar que la imagen se repita, debemos usar la propiedad CSS background-repeat: no-repeat;. Esto puede hacer que la imagen sólo se muestre una vez y la parte que no alcance a ser cubierta por la imagen, quede en blanco (o en el color de fondo que hayamos seleccionado). Para evitar que queden espacios en blanco podemos hacer que la imagen cubra todo el elemento deseado con las propiedades CSS background-size: cover; y background-attachment: fixed; las cuales deben usarse juntas para asegurarnos de que siempre se abarque todo el elemento. Esta combinación hace que la imagen conserve su relación de ancho y alto. Además, el elemento background-attachment: fixed; hace que la imagen se mantenga fija sin tomar en cuenta el scroll en la página, lo que puede dar un efecto de 3d cuando se utiliza en elementos HTML diferentes a body.

Otra forma de hacer que una imagen abarque toda la página es estirarla al tamaño entero de la página. Esta opción podría no ser la mejor en muchos casos, ya que la imagen perdería su relación de aspecto. Para ello se vuelven a utilizar las propiedades background-repeat: no-repeat; background-attachment: fixed; y adicionalmente se agrega background-size: 100% 100%;
