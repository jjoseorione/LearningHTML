Modificar enlaces con CSS

Descripción

En HTML es posible modficar de varias maneras la forma en la que se muestran los enlaces de una página. Por default en HTML todos los enlaces se muestran subrayados, de color azul cuando son links aún no visitados (que no se encuentran en el historial), de color morado cuando ya han sido visitados con anterioridad y de color rojo cuando son links activos (Un link es activo cuando es seleccionado dando click, esto se muestra solo por un momento).

Pseudoselectores

Debido a que todos los enlaces son siempre creados con la etiqueta <a>, se debe usar un pseudoselector para diferenciar entre enlaces no visitados, enlaces visitados, enlaces activos e incluso enlaces cuando tienen el mouse encima; de esta forma, cada tipo de enlace puede tener un estilo propio. Los peudoselectores usados para la etiqueta a son los siguientes:

  • :link Se utiliza para especificar enlaces que no han sido visitados.
  • :visited Se utiliza para especificar enlaces que ya han sido visitados con anterioridad.
  • :hover Se utiliza para especificar el estilo que tomará en enlace cuando tengan el mouse por encima.
  • :active Se utiliza para especificar enlaces activos.

Ejemplos

Ejemplo 1

Ejemplo de estilo de enlace que cambia de color según su estado: verde para no visitado, azul para visitado, rojo para hover y amarillo activo:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
a:link {
color: green;
background-color: transparent;
text-decoration: none;
}
a:visited {
color: blue;
background-color: transparent;
text-decoration: none;
}
a:hover {
color: red;
background-color: transparent;
text-decoration: underline;
}
a:active {
color: yellow;
background-color: transparent;
text-decoration: underline;
}
</style>
</head>
<body>

<a href="">Esto es un enlace</a>

</body>
</html>

Ejemplo 2

Ejemplo de estilo de enlace en forma de botón que cambia de color con hover y active:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
a:link, a:visited {
background-color: #f44336;
color: white;
padding: 15px 25px;
text-align: center;
text-decoration: none;
display: inline-block;
}
a:hover, a:active {
background-color: red;
}
</style>
</head>
<body>

<a href="">Esto es un enlace</a>

</body>
</html>
