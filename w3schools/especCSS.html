Formas de especificar un formato CSS

Descripción

Existen distintas formas de especificar el estilo de una página CSS. Si vamos de lo particular a lo general, la forma más básica sería poner estilo dentro de cada etiqueta que se usa en html, la siguiente forma consiste de un estilo general por página y por último, usando un archivo externo que puede ser usado en más de una página.

Usar estilos por etiqueta es de gran utilidad para cuando queremos dar estilo a una única sección de un documento HTML. Si queremos usar el mismo estilo más de una vez en un documento o un toda una web, esta forma no es lo más recomendado, ya que se tendría que especificar el estilo cada vez que es utilizado. Para usar estilos por etiqueta se debe hacer uso del atributo style="Instrucciones_CSS_aquí" en el elemento deseado.

Si queremos usar un estilo más de una vez en un mismo documento, lo mejor será declarar ese estilo de forma global para todo el documento. Para eso debemos de crear un bloque de código CSS al inicio del documento. Esto se logra con las etiquetas <style></style> dentro de la sección head del documento. Entre las etiquetas style se deben escribir las instrucciones CSS.

En caso de que se vayan a usar los mismos estilos de más de una página, lo mejor es crear un archivo externo que contenga todos los estilos. Este archivo debe tener sólo sentencias CSS (sin HTML) y debe tener extensión .css. Posteriormente, dicho atchivo debe ser convocado en nuestros documentos HTML en la sección head con la etiqueta <link rel="stylesheet" href="ruta_archivo_CSS">

Ejemplos

Ejemplo de CSS por etiqueta:

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>
<h3 style="color: blue; background-color: yellow;"> Texto azul con fondo amarillo </h3>
</body>
</html>

Ejemplo de CSS por documento:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<style>
h1{
color: blue;
background-color: yellow
}
</style> </head>
<body>
<h3> Texto azul con fondo amarillo </h3>
</body>
</html>

Ejemplo de CSS por arhivo externo invocado:

<html>
<head>
<title></title>
<meta charset="UTF-8">
<link rel="stylesheet" href="./styles.css"> </head>
<body>
<h3> Texto azul con fondo amarillo </h3>
</body>
</html>
