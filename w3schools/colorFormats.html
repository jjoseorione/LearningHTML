Formatos de color

Descripción

Existen 3 formas de describir un color en CSS, éstas son en rgb/rgba decimal, rgba hexadecimal y en hsl/hsla. Las siglas RGB y RGBA significan Red-Green-Blue y Red-Green-Blue-Alpha. HSL signfica Hue-Saturation-Alpha.

En rgb y rgba decimal escribimos la cadena rgb o rgba según corresponda, seguido de de un paréntesis que contenga los valores decimales de rojo, verde, azul y canal alpha (transparencia) de la forma rgba(rojo, verde, azul, apha). Los valores de cada color sólo pueden ir de 0 a 255. El canal apha debe tener un valor de entre 0 y 1.

En rgb hexadecimal debemos escribir un símbolo '#' seguido de dos dígitos hexadecimales (de 0 a F) para rojo, dos para verde y dos para azul con la forma #RRGGBB. De esta forma, el valor mínimo para cualquiera de los parámetros en 00 y el máximo es FF.

En hsl y hsla se utiliza el mismo formato que en rgb y rgba decimal usando la palabra hsl o hsla seguida de un paréntesis con los valores de matiz, saturación, brillo y canal alpha en la forma hsl(matiz, saturación, brillo). El valor de H puede ir de 0 a 360, ya que lo que hace es rodear todo el círculo cromáico. Los valores de S y L se expresan en porcentaje de 0% a 100%.

Ejemplo

Ejemplo de formatos de color:

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>
<div style="color: rgb(47,96,09); background-color:rgba(74,24,76,0.5);">
Este texto utiliza rgb(47,96,09) como color base y rgba(74,24,76,23) como color de fondo
</div>
<div style="color: #A9F2EB; background-color:#00C927;">
Este texto utiliza #A9F2EB como color base y #00C927 como color de fondo
</div>
<div style="color: #A9F2EB; background-color:#00C927;">
Este texto utiliza #A9F2EB como color base y #00C927 como color de fondo
</div>
<div style="color: hsl(183, 56%, 12%); background-color:hsla(263, 32%, 54%, 0.3);">
Este texto utiliza hsl(183, 56%, 12%) como color base y hsla(263, 32%, 54%, 0.3) como color de fondo
</div>
</body>
</html>
