Elemento <picture>

Descripción

El elemento de apertura y cierre <picture<</picture> tiene la utilidad de dar mayor flexibilidad a los desarrolladores para especificar imágenes. El elemento <pictue> encierra a uno o más elementos <source> y a un elemento img. Los elementos source especifican diferentes imágenes a través del atributo srcset="ruta_imagen" y cuándo es mejor utilizar dichas imágenes según el atributo media="mejor_caso", mientras que el elemento img da la última opción de imagen y además define los parámetros que tendrá la imagen que se mostrará finalmente. De esta manera, el navegador puede elegir cuál es la mejor imagen para mostrar; es decir, sólo se muestra un elemento entre los source y el img, y éste es el más adecuado según el atributo media de cada uno.

Ejemplo

Ejemplo de uso del elemento picture. Redimensiona la ventana para ver el resultado:

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>
<picture>
<source media="(min-width: 1000px)" srcset="../images/twin_towers1.jpg">
<source media="(min-width: 750px)" srcset="../images/twin_towers2.jpg">
<img src="../images/twin_towers3.jpg" style="width:100%;">
</picture>
</body>
</html>
