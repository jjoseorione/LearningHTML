Etiquetas mark, del, ins, sub, sup

Descripción

El atibuto mark es un elemento que sirve para resaltar en color fluorescente una porción de texto, como si se tratara de plumón marcatextos.

El atributo del muestra una porción de texto tachado por una línea.

El atributo ins muestra una porción de texto subrayada.

El atributo subs text muestra una porción de texto como subíndice.

El atributo sup muestra texto como superíndice.

Ejemplo

Ejemplo de código:

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>
Este texto está en color normal y <mark>este otro texto está resaltado</mark>. <del>Este texto está tachado</del> y <ins>este otro texto está subrayado</ins>. <sup>Este texto está como superíndice</sup> y <sub>este texto está como subíndice</sub>.
</body>
</html>
