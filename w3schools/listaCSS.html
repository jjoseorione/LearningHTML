Listas con CSS

Listas no ordenadas

Descripción

Con CSS es posible modificar varios aspectos de las listas, al grado de poder convertirlas en menús. Eso se verá más adelante, pero un aspecto básico es el de poder cambiar la viñeta que se muestra antes de cada elemento de la lista. Para ello se utiliza la propiedad CSS list-style-type:. Esta propiedad puede tomar los valores disc, circle. square y none. El que se muestra por default es disc.

Ejemplo

A continuación se muestran 4 listas no ordenadas. Cada una con una viñeta diferente

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>

Lenguajes de programación:
<ul style="list-style-type: disc;">
<li>C</li>
<li>Java</li>
<li>Python</li>
<li>PHP</li>
</ul>
Palos en el poker:
<ul style="list-style-type: circle;">
<li>Espadas</li>
<li>Diamantes</li>
<li>Corazones</li>
<li>Tréboles</li>
</ul>
Subgéneros del metal
<ul style="list-style-type: square;">
<li>Thrash Metal</li>
<li>Heavy Metal</li>
<li>Death Metal</li>
<li>Black Metal</li>
</ul>
Modelos de color:
<ul style="list-style-type: none;">
<li>RGB</li>
<li>CMYK</li>
<li>HSV</li>
</ul>

</body>
</html>

Listas ordenadas

Atributo type

Descripción

De la misma forma, también es posible cambiar el formato de ordenación en las listas ordenadas. Esto es posible lograrlo con el atributo type="numTipo", el cual puede tomar los valores 1, A, a, I y i. De esta forma, se sigue el formato marcado por ese tipo (1, 2, 3; A, B, C; a, b, c; I, II, III; i, ii, iii). El valor por default es 1.

Ejemplo

A continuación se muestran 5 listas ordenadas. Cada una con una forma de ordenación diferente

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>


</body>
</html>
