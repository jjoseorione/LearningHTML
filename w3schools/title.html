Atributo Title

Descripción

El atributo title puede usarse en cualquier elemento html. Sirve para mostrar una pequeña descripción de dicho elemento al poner el mouse encima.

Ejemplo

Se muestra un párrafo con una descripción

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>
<p title="Hola, esto es un párrafo">
Hola a todos. Este es un ejemplo del uso del atributo title. Pasa el mouse por encima de este párrafo para ver el resultado
</p>
</body>
</html>
