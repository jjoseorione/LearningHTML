Imagen con CSS

Descripción

Las imágenes también pueden ser modificadas con CSS. En este capítulo veremos únicamente dos propiedades CSS de imágenes: width, height y float. Las propiedades width y height, como su nombre lo indica, son para elegir las dimensiones de una imagen. Es más recomendable definir las dimensiones en estilo por elemento, debido a que una regla general en una hoja de estilo externa podría modificar crucialmente la forma en que se despliega una imagen. Por otra parte, el atributo float ayuda a desplegar una imagen de mejor forma al combinarla con texto al rededor.

Ejemplos

Ejemplo 1

Ejemplo de la diferencia entre usar siempre width y height en HTML y en CSS con una regla de etilo general previmente definida:

<html>

<head>
<style>
img
{
width: 100%;
}
</style>
<title></title>
<meta charset="UTF-8">
</head>

<body>

<img src="../images/img_girl.jpg" alt="Girl in a jacket" title="Girl in a jacket 1" width="500" height="600">
<img src="../images/img_girl.jpg" alt="Girl in a jacket" title="Girl in a jacket 2" style="width:500px; height:600px;">

</body>

</html>
