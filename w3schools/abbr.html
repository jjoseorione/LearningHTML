Etiqueta abbr

Descripción

La etiqueta abbr sirve para encerrar una abreviación. Se le agrega el atributo title para que se pueda visualizar el significado de dicha abreviación.

Ejemplo

Ejemplo de abbr:

<html>
<head>
<title></title>
<meta charset="UTF-8">
</head>
<body>
La <abbr title="Organización Mundial de la Salud">OMS</abbr> mitió al afirmar que el SARS-CoV2 no tenía potencial pandémico.
</body>
</html>

Vista previa:

La OMS mitió al afirmar que el SARS-CoV2 no tenía potencial pandémico.
