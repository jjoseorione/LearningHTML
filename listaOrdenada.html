Lista ordenada con etiqueta <ol>

Descripción

La etiqueta <ol> es bastante útil cuando debemos ordenar o listar una serie de datos. Es decir, despliega una lista de elementos enumerados. Consta de dos partes, una es con la etiqueta <ol>, la cual tiene apertura y cierre y encerrará a todos los elementos a enumerar. La otra parte es con la etiqueta <li>, la cual también cuenta con apertura y cierre y encerrará a cada elemento de la lista por separado. Ol significa ordered list y li significa list item. Recordemos además que los elementos HTML pueden anidarse, así que podemos utilizar una lista como elemento de una lista de nivel superior, y así tener listas anidadas. También podemos agregar énfasis a los elementos o convertirlos en un enlace.

Ejemplo

Si quisiéramos hacer una lista de los temas de este curso, desde el primero hasta el actual, tendríamos el siguiente código:

<ol>
<li>Introducción</li>
<li>Construcción de párrafos</li>
<li>Uso de títulos</li>
<li>Marcadores de énfasis</li>
<li>Uso de hipervínculos</li>
<li>Imágenes dentro de una página</li>
<li>Vínculos que abren de forma especial</li>
<li>Vínculos a un cliente de correo</li>
<li>Ancla llamada desde la misma página</li>
<li>Lista ordenada</li>
</ol>

Y el resultado sería el siguiente:

  1. Introducción
  2. Construcción de párrafos
  3. Uso de títulos
  4. Marcadores de énfasis
  5. Uso de hipervínculos
  6. Imágenes dentro de una página
  7. Vínculos que abren de forma especial
  8. Vínculos a un cliente de correo
  9. Ancla llamada desde la misma página
  10. Lista ordenada
