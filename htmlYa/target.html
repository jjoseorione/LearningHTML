Atributo "target" dentro de la etiqueta <a>

Descripción

La etiqueta <a> cuenta con el atributo "target", el cual permite indicar dónde será abierto el recurso al que se hace referencia. Los valores que puede tomar target son "_blank", "_top", "_self", "_parent" y "framename".

target="_blank"

Cuando target="_blank" se abre el recurso en una nueva pestaña

Ejemplo de target="_blank"

target="_top"

Cuando target="_top" se abre el recurso en toda la ventana. Esto tiene significado cuando la ventana está dividida en secciones, lo cual se verá más adelante. Ejemplo:

<a href="example_t01.html" target="_top">Entire window</a>

Ejemplo de target="_top"

target="_self"

Cuando target="_self" se abre el recurso en la misma sección donde se encuentra el enlace.

<a href="example_t02.html" target="_self">Same frame</a>

Ejemplo de target="_self"

target="_parent"

Cuando target="_parent" se abre el recurso en el recuadro padre del que abrió el enlace.

<a href="example_t03.html" target="_parent">Parent frame</a>

Cuando cargas "child.html" dentro del recuadro B, el recuadro padre de C es el recuadro B.

Ejemplo de target="_parent"

target="framename"

Cuando target="framename", donde freename es el nombre de un recuadro específico

<a href="example_t04.html" target="top">B frame</a>
<a href="example_t04.html" target="main">C frame</a>

Ejemplo de target="framename"

Nota importante

Los frames se encuentran actualmente en desuso. Se recomienda no crear páginas con uso de frames, ya que hacen la página menos accesible.
