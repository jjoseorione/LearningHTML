Entrada radio

Definición

La entrada radio sirve para seleccionar una y sólo una opción dentro de una lista mostrada, a diferencia de los checkbox que sirven para elegir una o más opciones. Para insertar un radio en un documento HTML se utiliza la etiqueta input con los atributos type="radio", name="nombre_variable" y value="valor_variable". Todos los radios de una misma lista deben llevar el mismo nombre de variable para que sólo uno de los valores sea asignado a dicha variable.

Ejemplo

Se requiere un formulario que muestre una lista con radios para que el usuario elija en qué intervalo de salario se encuentra. El código es el siguiente:

<form>
Seleccione en qué intervalo se encuentra su salario mensual: <br>
<input type="radio" name="salario" valor="1">$0.01 a $578.58 <br>
<input type="radio" name="salario" valor="2">$578.59 a $4 910.18 <br>
<input type="radio" name="salario" valor="3">$4 919.19 a $8 629.20 <br>
<input type="radio" name="salario" valor="4">$8 629.21 a $10$ 031.07 <br>
<input type="radio" name="salario" valor="5">$10 031.08 a $12 009.94 <br>
<input type="radio" name="salario" valor="6">$12 009.95 a $24 222.31 <br>
<input type="radio" name="salario" valor="7">$24 222.32 a $38 177.69 <br>
<input type="radio" name="salario" valor="8">$38 177.70 a $72 887.50 <br>
<input type="radio" name="salario" valor="9">$72 887.51 a $97 183.33 <br>
<input type="radio" name="salario" valor="10">$97 83.34 a $291 550.00 <br>
<input type="radio" name="salario" valor="11">$291 550.01 En adelante <br>
<input type="submit" value="Enviar">
</form>
