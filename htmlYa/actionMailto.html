Envío de datos por e-mail

Descripción

Una forma de obtener los datos de un formulario sin que sea necesario un lenguaje de programación es mandar los datos por correo electrónico. Para que esto sea posible debe usarse la palabra mailto en el atributo action de la siguiente forma: action="mailto:correo@deseado.com", además de agregar el atributo enctype="text/plain", el cuál le dice al formulario qué forma de codificación utilizar (en este caso, texto plano).

Ejemplo

Se requiere confeccionar una página que permita hacer un reclamo de reparaciones y se envíen los datos a una dirección de correo. Se debe ingresar nombre, dirección y una descripción del problema. El código sería el siguiente:

<form action="mailto:serpent-@hotmail.com?subject=Buzón de quejas" method="post" enctype="text/plain">
  <fieldset>
    <legend>Comuníquese con nosotros</legend>
    Nombre:
    <input type="text" name="nombre" size="30" placeholder="Ej. Eduardo" required>
    <br>
    Apellidos:
    <input type="text" name="apellidos" size="30" placeholder="Ej. Sánchez" required>
    <br>
    Dirección:
    <input type="text" name="direccion" size="40" placeholder="Calle, número, colonia, municipio, estado" required>
    <br>
    Comentario:
    <br>
    <textarea rows="5" cols="40" name="comentario" placeholder="Describa aquí el problema..." required></textarea>
    <br><br>
    <button type="submit">Enviar<img src="./sendMail.png" alt="save icon" width="15" height="15"></button>
  </fieldset>
</form>

Y éste sería el resultado:

Comuníquese con nosotros Nombre:
Apellidos:
Dirección:
Comentario:


Adicional: placeholder

Como puede notarse, el formulario anterior contiene dentro de los inputs una pequeña sugerencia de cómo puede ser llenado cada campo. Esto se logra con el atributo placeholder="textoGuía", donde textoGuía es el texto que se mostrará al usuario en color claro y desaparecerá en cuando éste empiece a escribir dentro del input.
