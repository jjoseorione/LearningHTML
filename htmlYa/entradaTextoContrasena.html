Entrada text y password

Definición

Como se dijo anteriorrmente, existen distintos tipos de entradas; dos de ellos son el tipo type="text" y el tipo type="password". Ambos son similares, pues ambos envían cadenas de texto. La diferencia radica en la forma en la que se muestra la información antes de ser enviada. El tipo texto se muestra de forma normal, mientras que en el tipo password se ocultan los caracteres escritos.

También cabe mencionar que existe otra variación del tipo texto, el cual es el tipo type="email". Este tipo de dato es útil cuando se requiere que la entrada sea un email, pues se asegura de que cuente con la estructura de una dirección de correo electrónico

Ejemplos

Se requiere crear un formulario en el que un usuario se registre en una plataforma en línea, para lo cuál debe introducir su nombre de usuario, su correo elecrónico y su contraseña. El código sería el siguiente:

<form>
Nombre de Usuario:
<input type="text" name="nombreUsuario">
<br>
Correo electrónico:
<input type="email" name="correo">
<br>
Contraseña:
<input type="password" type="contrasena">
<br>
<input type="submit" name="enviar">
</form>

Y éste sería el resultado.

Nombre de Usuario:
Correo electrónico:
Contraseña:

Para fines prácticos, sólo se mostrará el formulario, sin que éste envíe los datos a ninguna página. Para tal fin, de momento se prescindirá de los atributos "action" y "method".
