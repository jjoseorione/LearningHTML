Conceptos básicos de hojas de estilo CSS

Descripción

En las primeras versiones de HTML se comenzaron a introducir elementos con la finalidad de mejorar la presentación del contenido de la página. Así aparecieron elementos para definir las fuentes, tamaños, colores, alineamiento y texto en bloques, etc.

Luego se creó otra teconología que separa el contenido de la presentación; ésta tecnología son las hojas de estilo. Con esto quedó perfectamente definido el objetivo de HTML (disponemos de contenido) y CSS (definimos la presentación de ese contenido).

Veamos como primer concepto la sintaxis para aplicar color de frente y fondo a un elemento HTML mediante CSS. Esto se logra con el atributo style, el cual lo tienen todos los elementos de HTML para poder darles formato y estilo. En este primer acercamiento se usarán las palabras CSS color y background-color.

Ejemplo

Se desea crear una página en la que el texto tenga un color específico de fuente y de fondo. El código sería el siguiente:

<h1 style="color: #0000FF">Título con texto azul</h1>
<p style="color: #888888; background-color: #ffffdd">
Este texto tiene un color de fuente grisaceo y el fondo es color hueso, como el de los periódicos.
</p>

Y éste sería el resultado:

Título con texto azul
