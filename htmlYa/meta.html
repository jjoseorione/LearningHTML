Etiqueta <meta> dentro de la cabecera de la página

Definición

Antes hemos usado un elemento meta que es fundamental donde se indica el tipo de codificación de caracteres que usará la página, el cual es:

<meta charset="UTF-8">

La etiqueta meta define metadatos acerca de nuestro documento HTML; es decir, almacena datos de los datos, los cuales son usados por los navegadores para saber cómo desplegar o recargar una página HTML, por los buscadores para encontrar palabras clave, entre otras cosas.

La etiqueta meta debe de ir siempre dentro de la sección <head>. Veamos ahora otras propiedades de la marca meta y los valores más comunes con los que podemos inicializarlos.

Atributo name

Name esecifica el nombre de los metadatos. No existe ninguna especificación de la W3C que defina los valores posibles para el atributo name, pero existen algunos que son estándares de facto:

application-name
Se utiliza para especificar el nombre de una aplicación que el sitio web puede utilizar para desplegar su contenido. Es usado principalmente para aplicaciones móviles.
autor
Especifica el autor de la página
description
Sirve para darle una breve descripción a la página. Esta descripción se muestra en los resultados de búsqueda de algún motor de búsqueda y no debe rebasar 150 caracteres.
generator
Sirve para especificar el software utilizado para generar la página. No se utiliza si la página fue hecha a mano.
keywords
Especifica un conjunto de palabras clave, separadas por comas, que pueden ser utilizadas para ubicar la página en internet y para que sea encontrada por motores de búsqueda.
viewport
Sirve para controlar el viewport de la página, es decir la ventana visible.

Atributo http-equiv

Proporciona un encabezado HTTP para la información/valor del atributo content.Su principal uso es para refrescar la página después de cierto tiempo.

Atributo content

Sirve para especificar dentro de él el contenido asociado con el atributo name o el atributo http-equiv.

Ejemplos

<meta name="keywords" content="HTML, CSS, Javascript">
Define las palabras HTML, CSS y Javascript para que los motores de búsqueda relacionen la búsqueda con dichas palabras.

<meta name="description" content="Página de tutoriales gratuitos para HTML y CSS">
Especifica todo el texto asociado al atributo content como descripción de la página. De esta forma, cuando la página sea mostrada como resultado de una búsqueda, se mostrará esta pequeña descripción.

<meta name="author" content="John Doe">
Especifica a "John Doe" como autor de la página.

<meta http-equiv="refresh" content="30">
Hace que la página se actualice por sí sola cada 30 segundos.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Especifica que el tamaño de ventana debe ser igual de ancho que el ancho de la pantalla del dispositivo desde el que se observa y que se debe de conservar la escala. Esto es para que la página se vea de buena forma en todos los dispositivos
