Propiedades relacionadas a las fuentes en CSS

Descripción

Las propiedades relacionadas a las fuentes más importantes en CSS son las siguientes:

  • font-family Define la fuente que se aplicará al elemento HTML. Las fuentes usadas.
  • font-size Muestra un texto subrayado.
  • font-style Muestra un texto subrayado.
  • font-weight Muestra un texto subrayado.
