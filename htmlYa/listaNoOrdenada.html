Lista no ordenada con etiqueta <ul>

Descripción

Una lista no ordenada, como su nombre lo indica, es aquella que no sigue un orden específico. Normalmente este tipo de listas se redactan con el uso de viñetas en lugar de números. Para crear una lista no ordenada se utiliza la etiquera <ul> encerrando a toda la lista, mientras que para cada elemento separado se sigue usando la etiqueta <li>. Ul significa unordered list.

Ejemplo

Supongamos que queremos hacer una lista de los lenguajes que hemos aprendido hasta el momento. El código sería el siguiente:

<ul>
<li>C</li>
<li>C#</li>
<li>Java</li>
<li>Python</li>
<li>PHP</li>
<li>RPG</li>
</ul>

Y el resultado sería el siguiente:

  • C
  • C#
  • Java
  • Python
  • PHP
  • RPG
