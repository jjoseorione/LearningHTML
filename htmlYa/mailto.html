Uso de etiqueta <a> para envío de correos

Descripción

El elemento <a> permite direccionar un hipervínculo a un programa de envío de correos que tengamos configurado en nuestra computadora. Para esto, sólo se debe poner como atributo href la secuencia "mailto:direccion@deseada.com" sustituyendo con la dirección real a la que deseamos enviar el correo. También puede agregarse más información desde la etiqueta, como el asunto que deberá tener por defecto el correo a mandar, el cuerpo del mensaje, e incluso especificar envío de mail con copia y con copia oculta a otras direcciones. Esto se realiza agregando el separador '?', seguido de los apartados subject=Asunto del mensaje, body=Cuerpo del mensaje, cc=direccion@deseada.com, bcc=direccion@deseada.com. Los apartados anteriores deben ir separados por el caracter '&' dentro del atributo href.

Ejemplos

Ejemplo 1

Click aquí para enviar email.

El enlace anterior para enviar correo tiene el siguiente código:

<a href="mailto:jjoseorione@gmail.com">aquí</a>

Ejemplo 2

Click aquí para enviar email.

El enlace anterior para enviar correo tiene el siguiente código:

<a href="mailto:jjoseorione@gmail.com?subject=Curso HTML&body=Muy buenas tardes. Estoy interesado en el curso de HTML.&cc=serpent-@hotmail.com">aquí</a>
