Subida de archivos con entrada tipo file

Descripción

El input tipo file nos permite enviar un archivo al servidor. Nuevamente HTML sólo indica al navegador que debe enviar el archivo al servidor, pero debe haber en el servidor un programa que lo almacence en una carpeta de dicho servidor. La sintaxis es la siguiente:

<input type="file" name="nombreArchivo">

Usamos el atributo type que ya conocemos y el atributo name, el cual dará nombre al archivo que será subido al servidor.

Una cosa muy importante a tener en cuent cuando hacemos upload de archivos al servidor es inicializar la propiedad enctype del elemento form de la siguiente manera:

<form method="método" action="archivo" enctype="multipart/form-data">

De esta forma le indicamos al navegador que el formulario almacena uno o más archivos que deben ser enviados al servidor.

Atributos

La entrada tipo file puede utilizar los siguientes atributos:

  • accept: Sirve para especificar los tipos de archivos que pueden ser usados y así asegurar que el servidor recibirá sólo tipos permitidos. Para ello se deben listar y separar con comas los tipos de archivos deseados (pueden ser sus extensiones). Por ejemplo, el atributo accept=".jpg, .gif, .png, .tiff, .bmp" aceptará sólo esos formatos de imagen. También puede usarse un seleccionador global para elegir todas las extensiones de un tipo de archivo. Por ejemplo, el atributo accept="audio/*" permitirá todos los archivos de tipo audio.
  • capture: Sirve para especificar qué cámara usar para la captura del archivo (en caso de que dentro del atributo accept se incluya a la cámara). Los valores permitidos son capture="user", el cual permite usar la cámara frontal, y capture="environment", el cual permite usar la cámara trasera. En caso de que este atributo no esté específicado, se dispositivo del usuario será el que decida qué medio de entrada usar.
  • multiple: Permite al usuario elegir más de un archivo a la vez.

Ejemplo

Se requiere un formulario en el que un usuario suba su foto de perfil. El código sería el siguiente:

<form>
  Ingrese su nombre:
  <input type="text" name="nombre" size="30">
  <br>
  Seleccione una foto de perfil:
  <input type="file" name="fotoPerfil" accept="image/*">
  <br>
  <button type="submit">Guardar datos <img src="./save.png"
alt="save icon" width="15" height="15"></button>
</form>
