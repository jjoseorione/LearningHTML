Checkbox

Definición

Las cajas de chequeo o checkbox son una forma en la que el usuario puede seleccionar una o más opciones dentro de una lista mostrada. Se crean con la etiquera input y el tipo checkbox. Todas deben tener un nombre diferente en el atributo name, debido a que cada una será una variable booleana diferente que puede tomar el valor de verdadero o falso.

Ejemplo

Se requiere que el ususario seleccione los lenguajes de programación que domina, para lo cual se le muestra una lista con checkboxes. Se muestra el siguiente código de ejemplo:

<form>
  Seleccione los lenguajes de programación que domina:   <br>
  <input type="checkbox" name="lengC">C<br>
  <input type="checkbox" name="lengCpp">C++<br>
  <input type="checkbox" name="lengPy">Python<br>
  <input type="checkbox" name="lengJava">Java<br>
  <input type="checkbox" name="lengCs">Cs<br>
  <input type="checkbox" name="lengPhp">PHP<br>
  <input type="submit" name="Enviar">
</form>

Y el resultado sería el siguiente:

Seleccione los lenguajes de programación que domina:
C
C++
Python
Java
Cs
PHP
