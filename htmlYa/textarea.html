Entrada Área de Texto

Definición

El elemento de tipo textarea nos permite el ingreso de varias líneas, a diferencia del cuadro de texto (input type="text"). Es comúnmente utilizado cuando queremos ingresar un comentario de una longitud larga de caracteres. El texto que se escribe entre las etiquetas de apertura y cierre puede servir como texto de ejemplo o puede dejarse vacío.

Ejemplos

Se tiene el siguiente código HTML perteneciente a un formulario

<form>
  Ingrese su nombre:
  <input type="text" name="nombre">
  <br>
  Ingrese su correo
  <input type="email" name="correo">   <br>
  Escriba sus comentarios <br>
  <textarea name="comentarios" rows="5" cols="60">Escriba aquí sus comentarios </textarea>
  <br>
  <input type="submit" value="enviar">
</form>

Y el resultado es el siguiente:

Ingrese su nombre:
Ingrese su correo
Comentarios:
