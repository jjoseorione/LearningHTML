Lista de definiciones con etiqueta <dl>

Definición

Como su nombre lo indica, una lista de definiciones sirve para asociar un término con la definición del mismo. Como se vio anteriormente, las listas ordenadas y no ordenadas se muestran en el navegador con cada elemento en un renglón propio y con una sangría. La lista definiciones hace algo similar, pues muestra en la parte superior el concepto y debajo de él, con una sangría, se muestra en forma de párrafo la definición del concepto. Para las listas de definiciones se utilizan tres etiquetas:

  1. Etiqueta <dl>: Envuelve a toda la lista y significa definition list.
  2. Etiqueta <dt>: Envuelve el término a definir, significa definition term.
  3. Etiqueta <dd>: Envuelve la definición y significa definition description.

Ejemplo

Si quisieramos definir algunas de las etiquetas que hemos visto hasta el momento en este curso, el código que usaríamos sería éste:

<dl>

<dt>Etiqueta "br"</dt>
<dd>Permite crear un salto de línea en el texto de una página HTML, esto debido a que el texto que se escribe en una página HTML no muestra los saltos de línea que nosotros escribimos en el código HTML, sino que muestra todo de corrido.</dd>

<dt>Etiqueta "p"</dt>
<dd>Sirve para dar formato de párrafo a una sección de texto. Esto hace que exista un renglón vacío antes y después de cada párrafo. Es más conveniente usar la etiqueta de párrafo a la de salto de línea.</dd>

<dt>Etiqueta "hr"</dt>
<dd>Muestra una línea horizontal que sirve para separar dos secciones de una página. Por ejemplo, para separar una parte de texto simple de una parte interactiva.</dd>

</dl>

El resultado del código anterior sería el siguiente:

Permite crear un salto de línea en el texto de una página HTML, esto debido a que el texto que se escribe en una página HTML no muestra los saltos de línea que nosotros escribimos en el código HTML, sino que muestra todo de corrido.
Etiqueta "p"
Sirve para dar formato de párrafo a una sección de texto. Esto hace que exista un renglón vacío antes y después de cada párrafo. Es más conveniente usar la etiqueta de párrafo a la de salto de línea.
Etiqueta "hr"
Muestra una línea horizontal que sirve para separar dos secciones de una página. Por ejemplo, para separar una parte de texto simple de una parte interactiva.
