Un parrafo es una oración o conjunto de oraciones referentes a un mismo tema. Todo lo que encerremos entre las marcas <p> y </p> aparecerá separado con un espacio respecto al próximo párrafo. La p significa paragraph.

Dentro de un párrafo puede haber saltos de línea con la etiqueta <br>, pero es mejor y más correcto dar formato con la directiva de párafo que con la de salto de línea.

Para escribir en itálicas se utiliza la etiqueta <i>, cuyo significado es italic.

Para escribir en negritas se utiliza la etiqueta <b>, que significa bold

Para escribir subrayado se utiliza la etiqueta <u> que significa underline

Para escribir tachado se utiliza la etiqueta <s>, que significa strkethrough.
