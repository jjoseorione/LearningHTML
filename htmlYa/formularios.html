Formularios

Definición

Los formularios permiten que el visitante de una página pueda cargar datos en el servidor. Esto es, puede introducir, por ejemplo, sus datos para un registro, agregar comentarios, o cualquier cosa que requiere ser llenado con datos por parte del usuario.

HTML tiene únicamente la capacidad de crear formularios y no puede recuperar los datos enviados por el usuario ni de registrarlos en el servidor. Para ello se requiere de algún lenguaje de programación que se ejecute en el lado del servidor, como PHP, ASP, ASP.Net, JSP, NodeJS, etc.

Para crear un formulario se debe usar la etiqueta <form>, la cual debe tener apertura y cierre. Dentro del form se pueden crear distintos tipos de entradas, como líneas numéricas, líneas alfanuméricas, botones, editores de línea, cuadros de chequeo, radios de selección, etc.

Los formularios pueden hacer uso de distintos atributos. Los dos principales son action="páginaDeAcción" y method="MétodoDeAcción". La página de acción es aquella que verá reflejada la entrada; es decir, la página que utiliza los datos de entrada. El método, dicho de forma sencilla, define si los datos son privados (por ejemplo, una contraseña) o si pueden mostrarse en la página sin ningún problema.

Las entradas usan la etiqueta <input> y también pueden usar diferentes atributos. Los principales son type="TipoDeEntrada" y name="nombreVariable". El atributo type, como su nombre lo indica, se refiere a qué tipo de datos se le piden al usuario, mientras que el atributo name guarda el nombre que se le de a la variable en donde se guardaran los datos introducidos por el usuario. El atributo size define el tamaño de la ventana de texto que se mostrará para escribir la entrada y no límita el número de caracteres que puedan introducirse.

Ejemplo

Se creará un formulario únicamente con una entrada de texto y un botón de envío. El código será el siguiente:

<form action="formulario1.html" method="post">
Escriba su nombre:
<input type="text" name="nombre" size="20">
<br>
<input type="submit" name="enviar">
</form>

Y éste sería el resultado:

Escriba su nombre:

Nota: Sólo en este ejemplo se mostrará la recuperación de variables en otra página; en adelante sólo se mostrará la estructura del forulario y se prescinidirá de los atributos action y method; sin embargo, estos son imprescindibles en formularios reales.
