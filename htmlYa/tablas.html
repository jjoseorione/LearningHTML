Creación de tablas con etiqueta <table>

Descripción

El objetivo fundamental de las tablas es mostrar una serie de datos en forma ordenada, organizando en filas y columnas. Algo importante es que no usaremos las tablas para organizar la página completa, pues esto se hará posteriormente con hojas de estilo CSS.

Etiquetas <table>, <tr> y <td>

Para la creación de tablas se utilizan 3 etiquetas fundamentales en el siguiente orden:

  1. La etiqueta <table> encierra a toda la tabla, marcando el inicio y final de la misma
  2. La etiqueta <tr> sirve para iniciar una fila en la tabla, por lo que todo lo que se encierre en dicha etiqueta será una fila de la tabla. Tr significa table row
  3. La etiqueta <td> sirve para crear una celda de datos, las cuáles van encerradas en filas. Ts significa table data

Etiquetas <th> y <caption>

Adicionalmente se puede hace uso de más etiquetas; dos de ellas son las siguientes:

  • La etiquta <th> sirve para crear una celda de encabezado, la cual es distinta al resto de celdas de la misma fila o columna. Th significa table header. Por default la celda de encabezado se muestra en negritas y con texto centrado.
  • La etiqueta <caption> permite agregar una celda de título a la tabla, la cual se mostrará a lo largo de toda una fila.

Atributo border

Uno de los atributos que podemos usar tanto dentro de la etiqueta <table>, como en las etiquetas de celda es border="tamaño_borde", el cual, como su nombre lo indica, da el tamaño de borde de la celda o tabla.
