Lista desplegable Select

Descripción

El elemento select es una lista desplegable. Este elemento HTML nos permite seleccionar una opción una opción entre un conjunto. Veremos en el concepto próximo que, según cómo lo configuremos, podemos seleccionar varias opciones. Para crear un select se utiliza la etiqueta de apertura y cierre <select name="nombreVariable"> y entre las etiquetas de apertura y cierre select, se escriben las etiquetas de apertura y cierre <option value="valorVariable">. Entre las etiquetas de apertura y cierre option se escribe el texto que se mostrará en dicha opción. El campo value de la opción seleccionada se le asignará a la variable del campo name del select.

Ejemplo

Se creará un formulario de registro, para lo cual el usuario debera escribir su nombre y el idioma que desea aprender. El código será el siguiente:

<form>
Ingrese su nombre:
<input type="text" name="nombre" size="30">
<br>
Seleccione su idioma:
<select name="idioma">
<option value="1">Español</option>
<option value="2">Inglés</option>
<option value="3">Francés</option>
<option value="4">Portugués</option>
<option value="5">Japonés</option>
</select>
<br>
<input type="submit" value="Enviar">
</form>
