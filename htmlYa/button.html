Creación de botones con etiqueta button

Descripción

El elemento button es un control visual que se puede utilizar para sustituir los controles:

<input type="submit" value="Enviar">
<input type="reset" value="Borrar">

Una de las ventajas del elemento button es que podemos agregar imágenes dentro del botón. El elemento button tiene apertura y cierre y dentro del botón se encontrará todo lo que se escriba entre sus etiquetas. La sintáxis del elemento buttón es la siguiente:

<button type="tipo"> Contenido del botón </button>

Además, el elemento button puede conjuntarse con elementos de programación para que realicen distintos tipos de acciones.

Ejemplo

Se requiere un formulario en el que el botón de envío de formulario lleve una imagen alusiva a su función. El código sería el siguiente:

<form action="" method="">
  Entrada de datos:<br><br>
  Nombre:
  <input type="text" name="nombre">
  <br>
  Edad:
  <input type="number" name="edad">
  <br>
  Fecha de Nacimiento:
  <input type="date" name="fecha">
  <br>
  Correo electrónico:
  <input type="mail" name="correo">
  <br>
  <button type="submit">Guardar datos <img src="./save.png" alt="save icon" width="15" height="15"> </button>
</form>
