Agrupamiento de controles en formulario

Descripción

En HTML existe un elemento llamado fieldset, el cuál sólo tiene el objetivo de recuadrar y agrupar un conjunto de controles de un formulario.

Debemos encerrar todos los controles a agrupar entre las marcas de apertura y cierre <fieldset> y </fieldset>. Además, para agregar un título a este recuadro debemos agregar otro elemento HTML llamado legend, el cual también tiene apertura y cierre.

Ejemplo

Se creará un formulario que conste de dos partes, uno en el que se llenaran los datos personales del usuario y otro donde se guardarán sus datos laborales. El código sería el siguiente:

<form>
<fieldset>
  <fieldset>
    <legend> Datos Personales </legend>
    Apellido Paterno:
    <input type="text" name="apPat" size="15">
    <br>
    Apellido Materno:
    <input type="text" name="apMat" size="15">
    <br>
    Nombre (s):
    <input type="text" name="nom" size="30">
    <br>
    Fecha de Nacimiento:
    <input type="date" name="fechaNac">
  </fieldset>
  <fieldset>
    <legend> Datos Laborales </legend>
    Número de Seguridad Social:
    <input type="text" name="nss" size="11" minlength="11" maxlength="11">
    <br>
    RFC:
    <input type="text" name="rfc" size="13" minlength="12" maxlength="13">
    <br>
  </fieldset>
  <button type="submit">Guardar datos <img src="./save.png" alt="save icon" width="15" height="15"></button>
</fieldset>
</form>

Y éste sería el resultado:

Nombre (s)
