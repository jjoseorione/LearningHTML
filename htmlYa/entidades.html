Entidades

Definición

Una entidad en HTML es una cadena de texto que comienza con el símbolo ampersand (&) y termina con punto y coma (;). Las entiedades son frecuentemente utilizadas para mostrar carcateres reservados de HTML, ya que de otro modo serían confundidos con código html como sucede con los símbolos < y >, y también para mostrar caracteres invisibles, como los espacios en blanco.

Caracteres reservados

A continuación se muestran algunos de los caracteres que deben ser mostrados con entidades para que el analizador gramatical de html no los confunda como parte del código HTML

Caracter Entidad Descripción
& &amp; Puede ser interpretado por el navegador como el inicio de una entidad
< &lt; Puede ser interpretado como el inicio de una etiqueta
> &gt; Puede ser interpretado como el cierre de una etiqueta
" &quot; Puede ser interpretado como el inicio de un atributo dentro de una etiqueta

No breaking speace

Definición

Otra entidad muy utilizada es &nbsp;, el cual significa No breaking speace. Tiene dos usos, el primero es para añadir espacios extra al texto y que sean visibles en el documento HTML, ya que si únicamente agregamos muchos espacios en blanco en el código, el documento sólo mostrará un espacio; el segundo es, como su nombre lo indica, generar un espacio que no sea roto en dos renglones distintos. Esto último es útil cuando existen palabras que deben ir ligadas siempre (por ejemplo, unidades junto a su magnitud).

Otros caracteres

Existen otros caracteres que también pueden usarse como entidades en HTML. Por ejemplo, el caracter ©, puede escribirse directamente en el código HTML o bien puede escribirse de forma &copy; o de forma &#XA9;
