Entradas en formularios

Descripción

Cabe recalcar que existen muchos tipos de entradas (input) en html y es posible que existan nuevos cada vez que se realice una actualización en HTML. Los diferentes tipos de entradas facilitan el control de las mismas para asegurarnos de que el usuario introduzca un tipo de dato específico de forma correcta. Por ejemplo, sería incorrecto que en un campo para "edad" una persona escribiera caracteres no núméricos. Además, cada tipo de input tiene distintos atributos que también ayudan a controlar las variables. Poniendo nuevamente el ejemplo de "edad", sería ilógico que una persona pusiera un número muy grande, como mil, por ejemplo, para su edad en años, por lo que se debe elegir un número máximo.

    Atributos max y min

Para controlar los posibles valores de entradas cuantitativas como números, fechas u horas, disponemos de las etiquetas max="valor_máximo" y min="valor_mínimo". Esto nos permite establecer un rango dentro del cual debe especificarse la entrada.

Ejemplos

A continuación se muestran algunos ejemplos de inputs. Los atributos max y min se validan al presionar el botón de "Enviar datos".

Tipo número:
Código: <input type="number" name="edad" max="120" min="12">

Tipo fecha:
Código: <input type="date" name="fechaNac" min="1900-01-01" max="2005-01-01">

Tipo color:
Código: <input type="color" name="color">

Tipo hora:
Código: <input type="time" name="hora" min="07:00" max="23:00">

Tipo oculto: Este tipo de entrada, como su nombre lo indica, no se muestra al usuario. Un uso para este tipo de entradas es reutilizar una variable y dejarla como valor para una entrada oculta, de manera que el usuario no sea capaz de cambiarla
