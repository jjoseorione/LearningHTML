Elemento <pre>

Descripción

El elemento HTML <pre> nos permite mostrar toda la información que está contenida entre las marcas de comienzo y fin tal cual como lo hemos escrito, respetanto incluso espacios en blanco y saltos de línea.

Ejemplo

Se requiere una página que muestre una porción de código C respetando identación y saltos de línea: El código sería el siguiente:

<div style="background-color: #FFFFFF;font-family: serif;">
   <h2>Ejemplo de un query en SQL</h2>
   <pre>
      #include<stdio.h>
      int main(void)
      {
         printf("Hola mundo");
         getch();
         return 0;
      }
   </pre>
</div>

Y éste sería el resultado:

Ejemplo de un query en SQL

    				#include<stdio.h>
    				int main(void)
    				{
    				   printf("Hola mundo");
    				   getch();
    				   return 0;
    				}
