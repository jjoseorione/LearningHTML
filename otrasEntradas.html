Entradas en formularios

Descripción

Cabe recalcar que existen muchos tipos de entradas (input) en html y es posible que existan nuevos cada vez que se realice una actualización en HTML. Los diferentes tipos de inputs facilitan el control de las mismas para asegurarnos de que los usuario introduzcan un tipo de dato específico de forma correcta. Por ejemplo, sería incorrecto que en un campo para "edad" una persona escribiera caracteres no núméricos. Además, cada tipo de input tiene distintos atributo que también ayudan a controlar las variables. Poniendo nuevamente el ejemplo de "edad", sería ilógico que una persona pusiera un número muy grande, como mil, por ejemplo, para su edad en años, por lo que se debe elegir un número máximo.

Ejemplos

A continuación se muestran algunos ejemplos de inputs. Se prescindirá del elemento form para mostrar únicamente los inputs.

Tipo número:
Código: <input type="number" name="edad">

Tipo fecha:
Código: <input type="date" name="fechaNac">

Tipo color:
Código: <input type="color" name="color">

Tipo hora:
Código: <input type="time" name="hora">

Tipo oculto: Este tipo de entrada, como su nombre lo indica, no se muestra al usuario. Un uso para este tipo de entradas es reutilizar una variable y dejarla como valor para una entrada oculta, de manera que el usuario no sea capaz de cambiarla
